# golang 中关于数组的相关特性

* 值类型 
> 作为参数传递时 不能修改其值 只是一种拷贝 而且还需要特别注意的就是 不同长度的数组类型不一样 
```golang
package main 
import (
	"fmt"
)

func print(arr [3]int) {
	arr[0] = 100
}

func main() {
	arr1 := [3]int{1, 2, 3}
	print(arr1)
}

//结果打印出来的应该仍然还是1，2，3
```

* 如果想要改变的话 需要使用指针传递作为参数 但是常见的是使用 slice 切片 自动可以修改原来的数据

* 数组类型跟长度有关系 可以定义[...]int 让编译器自己去判断数组长度
